Gestión de Backup

Realizar backup

Crea una copia de seguridad completa del estado actual de la base de datos. Este proceso puede tardar unos minutos dependiendo del tamaño de los datos.

info

Realiza copias de seguridad periódicas para garantizar la integridad de tus datos. Las copias se guardan con la fecha y hora de su creación

Restaurar backup

Selecciona una copia de seguridad de la lista para restaurar la base de datos a ese estado. Esta acción reemplazará los datos actuales.

warning

Advertencia: La restauración de una copia de seguridad reemplazará todos los datos actuales. Esta acción no se puede deshacer.
